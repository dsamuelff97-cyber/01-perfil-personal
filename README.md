# Personal Profile Page

Static personal profile page built with semantic HTML5 as the first project of my fullstack developer learning path.


## Live Demo
[View live site](https://dsamuelff97-cyber.github.io/01-perfil-personal/)

## Technologies
- HTML5
- Semantic structure

## What I practiced 
- Semantic tags: header, main, section, footer, nav
- Accessibility with alt attributes
- Anchor links between sections
- Document structure with head, meta tags and viewport

## About this project
This is the first project of my portfolio. It presents who I am, my skills and my contact information using correct and accessible HTML structure, without any CSS or Javascript yet.
